Quick Start

Get up and running in under 5 minutes

1

Set Up Environment

Clone the repository and configure your API key

# Clone and setup
git clone <repository-url>
cd ai-coder

# Set your Gemini API key
export GEMINI_API_KEY='your-api-key-here'
2

Start the Backend

Install dependencies and run the server

cd backend
pip install -r requirements.txt
python main_enhanced.py

# Server starts at http://localhost:8000
3

Start the Frontend (Optional)

Launch the interactive chat interface

cd frontend
npm install
npm start

# UI available at http://localhost:3000
4

Generate Your First App

Describe what you want to build in natural language

curl -X POST http://localhost:8000/api/chat/message \
  -H "Content-Type: application/json" \
  -d '{"message": "Build a todo app with user authentication"}'

API Reference

Complete API documentation for integrating with your applications

POST /api/chat/message Main chat endpoint - streaming SSE response

Send a natural language description and receive a streaming response with generated files.

Request Body
{
  "message": "string",           // Required: Description of the app
  "conversation_id": "string"  // Optional: Continue existing conversation
}
Response (SSE Stream)
// Event types:
{ "type": "started", "conversation_id": "..." }
{ "type": "features_proposed", "data": { "features": [...] } }
{ "type": "architecture_designed", "data": { "project_type": "..." } }
{ "type": "file_generated", "data": { "filepath": "...", "content": "..." } }
{ "type": "code_generated", "data": { "total_files": 35 } }
{ "type": "message_end" }
POST /api/v1/generate Direct generation - non-streaming JSON response 
// Request
{
  "description": "Build a REST API for a blog with posts and comments",
  "constraints": { "framework": "FastAPI" }
}

// Response
{
  "status": "success",
  "files": [...],
  "architecture": {...},
  "metadata": { "total_files": 25, "mode": "batch" }
}
GET /api/chat/health Health check and system status 
{
  "status": "healthy",
  "version": "6.0.0-batch-streaming",
  "mode": "batch",
  "improvements": {
    "api_calls_reduction": "86%",
    "cost_reduction": "93%"
  }
}
POST /api/execute Execute generated application

Save files to disk, install dependencies, and start the dev server. 

{
  "files": [...],              // Generated files array
  "conversation_id": "..."    // Optional session ID
}
POST /api/download Download project as ZIP

Returns a ZIP file containing all generated project files.

GET /api/v1/usage Get LLM usage statistics
